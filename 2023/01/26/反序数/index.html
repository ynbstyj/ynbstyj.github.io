反序数

枚举 清华大学

描述

设N是一个四位数，它的9倍恰好是其反序数（例如：1234的反序数是4321）
求N的值

输入描述：

程序无任何输入数据。

输出描述：

输出题目要求的四位数，如果结果有多组，则每组结果之间以回车隔开。

#include <bits/stdc++.h>
using namespace std;

int Reverse(int x) {
    int a = x % 10;
    int b = x / 10 % 10;
    int c = x / 100 % 10;
    int d = x / 1000;
    int revx = a * 1000 + b * 100 + c * 10 + d;
    return revx;
}

int main() {
    for (int i = 1000; i <= 9999; i++) {
        if (i * 9 == Reverse(i))
            printf("%d\n", i);
    }
    return 0;
}
