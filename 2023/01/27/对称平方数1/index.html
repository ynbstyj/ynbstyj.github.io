对称平方数1

枚举 清华大学

描述

打印所有不超过256，其平方具有对称性质的数。如2，11就是这样的数，因为22=4，1111=121。

输入描述：

无任何输入数据

输出描述：

输出具有题目要求的性质的数。如果输出数据不止一组，各组数据之间以回车隔开。

#include <bits/stdc++.h>
using namespace std;

int Reverse(int x) {
    int revx = 0;
    while (x) {
        revx *= 10;
        revx += x % 10;
        x /= 10;
    }
    return revx;
}

int main() {
    for (int i = 0; i < 256; i++) {
        if (i * i == Reverse(i * i))
            cout << i << endl;
    }
    return 0;
}
